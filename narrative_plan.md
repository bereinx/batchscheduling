# Refinery Batch Scheduling Executive Deck Plan

## Audience
- 정유 공장 운영팀
- 생산계획/스케줄링 담당자
- 내부 보고/설명용 청중

## Objective
- 현재 구축한 배합 스케줄링 시스템의 목적, 접근 방식, 구현 범위, 활용 가치를 짧고 명확하게 설명한다.

## Narrative Arc
1. 왜 이 문제가 어려운가
2. 어떤 방식으로 풀었는가
3. 지금 시스템이 실제로 무엇을 보여주는가
4. 현업에서 어떤 가치가 있는가
5. 앞으로 어떻게 확장할 수 있는가

## Slide List
1. Title / one-line value proposition
2. Operational complexity and planning challenge
3. Algorithm architecture
4. What the current tool delivers
5. Practical value and roadmap

## Visual System
- 16:9 landscape
- 차분한 refinery dashboard 톤
- 블루-그레이, muted teal, soft orange accent
- 과한 gradient 없이 matte panel style
- 텍스트는 editable object로 구성

## Asset Plan
- 이번 deck은 외부 이미지 없이 editable shape 기반
- 간단한 process diagram, cards, timeline, matrix를 shapes/text로 구성

## Editability Plan
- 모든 제목, 본문, 강조 문구, 카드, 표 라벨을 editable text box로 생성
- 발표자가 직접 수치/문구를 수정할 수 있도록 PPT native object 중심으로 구성
